# Многопоточная сумма в C++

Этот проект демонстрирует **однопоточное и многопоточное суммирование** большого массива целых чисел на C++. Используется `std::thread` из стандарта C++11 и сравнивается время выполнения между одним и несколькими потоками.

---

## Возможности

- Генерация вектора из **10 миллионов случайных чисел** (0–99)  
- Вычисление **суммы всех элементов в одном потоке**  
- Вычисление **суммы с использованием нескольких потоков** (по умолчанию: 4 потока)  
- Измерение и сравнение времени выполнения  
- Демонстрация **основ многопоточности и лямбда-функций** в C++  

---

## Требования

- C++11 или выше  
- Компилятор с поддержкой потоков (например, g++, clang++, MSVC)  
- Совместимо с Windows, Linux и macOS  

---

## Как собрать и запустить

### Используя g++

```bash
g++ -std=c++11 -o multithread_sum main.cpp
./multithread_sum
```
Используя Visual Studio / VSCode
Откройте папку проекта в VSCode.

Убедитесь, что установлен C++ компилятор (MSVC, MinGW и др.).

Соберите main.cpp и запустите исполняемый файл.

## Пример вывода
Однопоточная сумма: 499987654.

Время (1 поток): 0.42 сек.

Многопоточная сумма: 499987654

Время (4 потоков): 0.18 сек.

Время выполнения может отличаться в зависимости от мощности компьютера.

## Примечания
Количество потоков можно изменить, изменив переменную numThreads в main.cpp.

Программа использует std::setlocale(LC_ALL, ""); для корректного отображения русских символов в консоли.

Многопоточность ускоряет работу с большими массивами, но прирост зависит от количества ядер процессора и загрузки системы.
